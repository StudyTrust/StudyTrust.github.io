SharePoint List Column Reference — Hand to Simon Letsela
Create 5 lists in the new SharePoint site (studietrust.sharepoint.com/sites/StudyTrustPersonnel). Use exact column names. Title exists automatically — do not recreate it.

PersonnelRegister

Title Text (built-in — auto-set to FirstName + LastName) · EmployeeNo Text · FirstName Text · LastName Text · KnownAs Text · DateOfBirth Date · Gender Choice: Male, Female, Other · Race Choice: African, Asian, Coloured, Indian, White, Foreign National · Disabled Yes/No · Nationality Choice: South African, Foreign National · Cluster Choice (all 9 clusters) · JobTitle Text · JobGrade Text · PayrollLevel Text · OccupationalLevel Choice: Top Management, Senior Management, Professionally Qualified, Skilled, Semi-Skilled, Unskilled · CostCentre Text · EmployeeType Choice: Permanent, Fixed Term, Intern, Part-Time · EmploymentType Choice: Full-Time, Part-Time, Contract · ContractType Choice: Permanent, Fixed Term, Internship · DateEngaged Date · ContractEndDate Date · LeaveGroupHolidays Text · LeaveGroupWorkDays Text · LeaveGroupEntitlement Number · ManagerName Text · WorkEmail Text · MedicalAid Yes/No · ForeignNational Yes/No · LastPayrollUpdate Date · Active Yes/No (default: Yes) · Status Choice: Active, Archived

PersonnelAlerts

Title Text (built-in = alert title) · AlertType Choice: Contract Expiry, Work Permit Expiry, Missing Document, Youth Threshold, Onboarding Incomplete, Performance Overdue, Medical Aid Missing, Internship End, Conflict of Interest Overdue, Other · EmployeeNo Text · EmployeeName Text · Cluster Text · TriggerDate Date · DueDate Date · Priority Choice: High, Medium, Low · Visibility Choice: HR Only, HR and Cluster Manager, Employee and Cluster Manager · AssignedTo Text · Status Choice: Active, Resolved, Dismissed · ResolutionNotes Note · ResolvedDate Date

PersonnelIR

Title Text (built-in = case number, format IR-YYYY-NNN) · CaseType Choice: Disciplinary, Grievance, Incapacity, Appeal · Category Choice: Misconduct, Poor Performance, Attendance, Substance Abuse, Insubordination, Other · EmployeeNo Text · EmployeeName Text · JobTitle Text · Cluster Text · JobGrade Text · PayrollLevel Text · DateReported Date · ReportedBy Text · HearingDate Date · CurrentStage Choice: Investigation, Hearing, Outcome, Appeal, Closed · Outcome Choice: Pending, Guilty, Not Guilty, Withdrawn · Sanction Choice: None, Written Warning, Final Written Warning, Suspension, Dismissal · CaseNotes Note · Status Choice: Open, Pending, Closed

PersonnelOnboarding

Title Text (built-in = EmployeeNo) · EmployeeNo Text · EmployeeName Text · Cluster Text · StartDate Date · ITSetup Yes/No · InductionComplete Yes/No · PolicyAcknowledgement Yes/No · ContractSigned Yes/No · Review30Date Date · Review60Date Date · Review90Date Date · ProbationEndDate Date · ProbationOutcome Choice: Pending, Confirmed, Extended, Terminated · ManagerSignOff Yes/No · OverallStatus Choice: In Progress, Completed, Escalated

PersonnelCOI (Conflict of Interest declarations)

Title Text (built-in = EmployeeNo + Year) · EmployeeNo Text · EmployeeName Text · DeclarationYear Text · HasConflict Choice: Yes, No · ConflictDetails Note · SubmittedDate Date · SubmittedBy Text (email)
Permissions: Break inheritance on PersonnelIR and PersonnelCOI lists. Grant access only to Dashboards - Governance-Admins and Dashboards - HR-Authorised-Personnel. All other groups should have no access to these two lists.
